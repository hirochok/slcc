# slcc

Simulation code for the paper "Behavioral evolution under social learning with complex contagion."

# Games

The code runs simulations for three game types:
- 0: Donation game
- 1: Snowdrift game
- 2: Coordination game

### Donation Game

\[
\begin{pmatrix}
b - c & -c \\
b & 0 
\end{pmatrix}
\]

### Snowdrift Game

\[
\begin{pmatrix}
b - \frac{c}{2} & b - c \\
b & 0 
\end{pmatrix}
\]

### Coordination Game

\[
\begin{pmatrix}
a & 0 \\
0 & b 
\end{pmatrix}
\]
